Ultimos Projetos

Site Para História De Um Tatuador

O site "Felipe Lacroes Tattoo" apresenta o trabalho do tatuador Felipe Lacroes, especializado em tatuagens realistas. Com mais de seis anos de experiência, Felipe dedica-se a transformar a pele em verdadeiras obras de arte, capturando cada detalhe com precisão. Seu foco é criar tatuagens que impressionam e contam histórias, utilizando técnicas e equipamentos de ponta para garantir que cada arte seja única e exatamente como o cliente imaginou.

Todo-List

Uma aplicação de lista de tarefas desenvolvida com React.js e Vite, com foco em performance e simplicidade. Funcionalidades: Adicionar, concluir e excluir tarefas. Interface responsiva e intuitiva. Esse projeto reforça conceitos de React, como hooks (useState, useEffect), e boas práticas na criação de componentes.

Protótipo para Hamburgueria

O site "Dev Burger" é uma aplicação web de cardápio para uma hamburgueria fictícia, desenvolvida utilizando HTML, CSS e JavaScript puro. Ele demonstra habilidades como manipulação dinâmica do DOM para exibir itens do cardápio, cálculo de totais em tempo real ao adicionar itens ao carrinho, alternância de temas (modo claro e escuro), e design responsivo. O projeto reflete domínio no uso de JavaScript para criar experiências interativas e funcionais.
